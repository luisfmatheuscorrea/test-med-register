#
## Instalação do Projeto
Para instalar, clone o projeto e dentro do diretório principal do projeto dê:
```
yarn
```
ou
```
npm i
```

### Compilar e rodar o projeto
Para rodar o projeto use:
```
yarn serve
```
ou
```
npm run serve
```

### Pronto!
